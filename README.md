# Brief JS — Sunu-idée

## Aperçu
Ce dépôt contient une petite application front-end en HTML/CSS/JavaScript. C'est un prototype léger destiné à démontrer une interface web statique et les interactions JavaScript associées.

## Lien de déploiement
- Site en ligne : https://sunu-id-e.vercel.app/

## Fonctionnalités
- Pages statiques servies depuis `index.html`.
- Styles simples dans `style.css`.
- Comportements et logique côté client dans `app.js`.

## Technologies
- HTML
- CSS
- JavaScript

## Exécution locale
Vous pouvez ouvrir `index.html` directement dans votre navigateur, ou lancer un serveur local pour éviter les limitations des pages locales.


## Structure du projet
- `index.html` — point d'entrée de l'application.
- `app.js` — scripts JavaScript.
- `style.css` — styles.
- `images/` — ressources graphiques.
